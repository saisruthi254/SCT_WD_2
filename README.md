# ⏱️ Stopwatch Web Application

This is a simple and interactive Stopwatch Web Application built using HTML, CSS, and JavaScript as part of my internship at SkillCraft Technology.

## Features
- Start, Pause, and Reset functionality  
- Lap time tracking  
- Real-time time updates  
- User-friendly interface  

##  Technologies Used
- HTML  
- CSS  
- JavaScript  

##  How It Works
- Uses JavaScript `setInterval()` to track time  
- Updates the display dynamically using DOM manipulation  
- Allows users to record lap times and manage the timer easily  

##  Learning Outcomes
- Gained hands-on experience with DOM manipulation  
- Learned event handling in JavaScript  
- Improved problem-solving and logic-building skills
